For use this project, you can run this command:
#    -php artisan serve # Only for the laravel project
For use the vite server, run the next command:
#   -npm run dev
inside the root of the project.

In case that doesn´t work, try to download Herd by the next hyperlink:
https://herd.laravel.com/download/latest/windows
 
If you want to work with some data inside the project, the only command you should have to run is:
## -php artisan db:seed  (Also a this project has a SQLite Database)
## -php artisan migrate:fresh --seed *In case you are working with artisan cli tools, more updated.
